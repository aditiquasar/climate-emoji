# 🌤️ Weather Emoji Converter

A simple Python program that converts weather descriptions into emojis using match-case statements.

## 📋 Description

This CLI tool takes a weather condition as input and returns the corresponding emoji. Perfect for beginners learning Python's match-case syntax and input handling.

## ✨ Features

- Accepts user input for weather conditions
- Case-insensitive input handling
- Returns appropriate emoji for each weather type
- Includes default fallback emoji

## 🚀 Getting Started

### Prerequisites
- Python 3.10+ (for match-case support)
